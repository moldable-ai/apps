Plan and ship work in projects, boards, and tasks that stay entirely on your machine. Tasks gives you a Linear-style tracker for personal and side projects, with no account, no sync server, and no data leaving your workspace.

**Highlights**

- Organize work into projects with three-letter keys and per-task numbers (like TES-1), each with its own summary, website, and labels
- Switch between a list view and a drag-friendly Kanban board across backlog, open, in progress, completed, and closed columns
- Set priorities (none/low/medium/high), apply color-coded labels, and filter tasks down to exactly what you need
- Write rich task descriptions and acceptance criteria, drop markdown comments, and attach files saved to local disk
- Keep a per-project Readme tab alongside your tasks for context, specs, or notes
- Comes with a full app API so your Moldable AI assistant can create, search, update, and triage projects and tasks for you

Everything is stored locally through Moldable's workspace storage and connects to no external services, so it is fast, private, and yours. Install it in one click and start tracking work in seconds.
