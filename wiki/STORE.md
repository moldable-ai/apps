Wiki is a Markdown-first knowledge base where you write notes, link them with `[[wiki links]]`, and watch the connections build into a navigable web of everything you want to remember. It runs entirely on your machine, so your notes stay yours.

**Highlights**

- Write and edit in a real Markdown editor with `[[wiki links]]`, embedded images, video, and audio, plus a live read view
- A side inspector surfaces outline, backlinks, unlinked mentions, and broken links so you can see how every note connects
- Typed frontmatter properties (text, list, tags, number, toggle, date, datetime) you can add, rename, reorder, and edit inline
- Built-in note templates for meetings, research, project briefs, decision records, people, and media logs, plus one-click daily notes
- Organize with drag-and-drop folders, full-text search, vault bootstraps (knowledge base, project, journal, health), and a trash with restore
- Run multiple vaults, switch fonts (including OpenDyslexic), and jump anywhere through the command palette

Local-first and self-contained, with no external accounts or services to connect. Install it for a fast, link-aware home for your notes that stays in sync and entirely under your control.
