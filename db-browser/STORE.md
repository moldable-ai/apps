A local-first workspace for your PostgreSQL databases: connect, explore schemas, run queries, and build dashboards without sending your credentials anywhere.

**Highlights**

- Manage multiple Postgres connections with environment labels (local, staging, production) and policy modes that mark each as read-only, write, or admin
- Browse schemas, tables, and views with fast search, then inspect columns and drill into individual rows
- Write SQL in a real editor with formatting and multi-statement execution, view results in a resizable grid, and revisit past runs from query history
- Export query results to CSV or JSONL, and import rows back into a table
- Build dashboards with line, area, bar, pie, donut, scatter, bubble, number, and table charts driven by your own read-only SQL
- Includes an agent API for schema lookups, EXPLAIN, and bounded read-only queries that never exposes connection credentials

Connects to your PostgreSQL databases through Moldable's aivault, so credentials stay secured and out of reach of agents. Install it to keep a fast, safe database client right alongside the rest of your tools.
