Explore your database without leaving Moldable. DB Browser connects to PostgreSQL and lets you inspect schemas, tables, and views — and run bounded, read-only queries safely.

**Highlights**

- Connect to PostgreSQL with saved connections
- Object browser for schemas, tables, views, and column details
- Saved SQL editor tabs with formatting and query history
- Read-only, bounded query results in a clean table
- Exposes safe APIs so Moldable can help you read schema and edit SQL
