A real IDE, right inside Moldable. Code is a lightweight, VS Code–like editor for exploring and editing local codebases — with a file tree, multi-tab editing, an integrated terminal, and a live browser preview.

**Highlights**

- File tree navigation with create, rename, and delete
- Multi-tab editor with Monaco syntax highlighting
- Integrated terminal and live browser preview for web projects
- Command palette for quick actions; remembers your open tabs
- Gives the Moldable agent rich context about your project
