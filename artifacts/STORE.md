Turn a chat into something worth publishing. Artifacts lets you generate beautiful, self-contained web pages and slide decks — dashboards, landing pages, product specs, charts, data stories, games, 3D scenes, and presentations — entirely from chat, then share them with a single public link.

**Highlights**

- Two kinds of artifact, one canvas: a **page** (a full, scrolling, responsive web page you fully control with HTML/CSS/JS) or a **deck** (a 16:9 slide presentation with smooth motion and an optional app-like runtime).
- Interactive decks without extra UX: calculators, filters, sortable tables, live charts, games, embeds, and click-to-build sequences work inside ordinary slides; static decks remain the default.
- Chat-first authoring: ask the assistant to create, surgically edit, restyle, or rebuild an artifact and it drives everything through the app's rich RPC — exact string edits via `artifacts.page.text.replace { oldString, newString }` / `artifacts.slides.text.replace { oldString, newString }`, whole-field page updates via `artifacts.page.set`, and deck structure via slide operations.
- A studio-grade template library to start from: a live analytics dashboard, a financial report, a SaaS landing page, a product spec, a startup one-pager, a visual case study, a generative 3D scene, an arcade game, plus carried-over presentation decks.
- Hand-rolled, library-free visuals: SVG and canvas charts with tabular numerals, scroll-reveal animation, parallax, and a `--scroll` progress variable for scroll-driven effects.
- Built-in image generation: create on-brand imagery from a prompt (or remix an existing image) and drop it straight into your artifact.
- One-click publish to a public, unlisted Moldable Artifacts link — a self-contained static page (HTML/CSS/JS + images) that works on any device, with republish, unpublish, and version history.
- Fully local: each artifact is plain data in your workspace, cleanly keyed so it's easy to manage and delete; nothing leaves your machine until you choose to publish.

Install it to turn a prompt into a polished, shareable creation.
