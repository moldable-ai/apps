# Workshop — style guide

Energetic, hands-on facilitation deck for live workshops, trainings, and
sprints. The bundled sample is a complete 18-slide design-thinking session
(run of show, exercises with timing, share-back, reflection) — tailor the
challenge, exercises, durations, and brand.

- Palette: warm paper `#fffdf7`, charcoal ink `#1c1917`, electric orange
  `#ea580c` (the one accent), muted stone `#78716c`. Type: display
  **Space Grotesk**, body **Inter** (both Google Fonts).
- Imagery: `workshop-cover.jpg` (overhead hands + orange stickies — cover and
  close), `workshop-fig.jpg` (activity close-up — the Exercise 2 split).
  Section breaks are clean paper `.divider`s with a bold orange rule.
- Bespoke decoration:
  - `.exbadge / .exbadge-n / .exbadge-k / .exbadge-t` — big circular numbered
    exercise badge; lead every exercise slide with it.
  - `.chip` (duration / detail) and `.chip.solid` (the active timer) — group in
    a `.chiprow`. Pair a solid timer with a few outline detail chips.
  - `.materials / .mat / .mat-box` — boxed-tick checklist for kit and takeaways.
  - `.yourturn / .yourturn-tag / .yourturn-text` — dashed orange callout for a
    live prompt the room acts on now.
  - `.agenda / .ag-row / .ag-time / .ag-name` — run-of-show timeline; each row
    ends in a duration `.chip`.
  - `.actcard*` — orange-top-rule cards for the framework flow and share-back.
  - `.divider*` — section breaks.
- Shared used: `.flow` (framework diagram), `.steps` (exercise instructions),
  `.checks` (ground rules, takeaways), `.bars` (confidence reflection),
  `.table` (next-steps plan), `.split.reverse` (Exercise 2 figure), `.quote`,
  `.runner` footer on every content slide.
- Arc: cover → today's goal → agenda & timing → ground rules → §Warm-up →
  ice-breaker (your-turn) → the framework (flow) → §Build → Exercise 1
  (badge + steps + timer) → Exercise 2 (badge + image split) → share-back
  (cards) → reflection (bars) → §Wrap → takeaways (checks) → resources &
  next steps → participant quote → close.
- Density notes: keep energy high and timing explicit. Every exercise gets a
  badge and at least one timer chip; never bury an instruction in prose — use
  `.steps`. One idea per slide; let the orange do the shouting, not the layout.
