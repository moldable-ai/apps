A focused, local-first Git client for staging, diffing, committing, and shipping changes without leaving Moldable. It pairs a fast diff viewer with AI that writes your commit messages and reviews your changes before you push.

**Highlights**

- Stage and commit across repos with multi-select, range-select, and a live status view that auto-refreshes every few seconds
- Generate a commit summary and body from your selected changes with one click, then commit, commit-and-push, or commit-and-open-PR
- Run an AI code review on selected files that returns severity-tagged findings you can copy, dismiss, or hand off straight into Moldable chat to fix
- Browse a syntax-styled diff for any changed file or past commit, including inline image previews and infinite-scroll commit history
- Open a GitHub pull request directly: it detects your remote, creates a branch if needed, drafts the PR title and body, and opens it
- Right-click to discard changes, add a file to .gitignore, reveal in Finder, or open it in a detected editor (including other Moldable apps), plus undo an unpushed commit

AI runs through Moldable's built-in model server, so there are no API keys to manage and no external services to connect.

Worth installing if you want everyday Git work in one tidy window, with the tedious parts (commit messages, reviews, PRs) handled for you.
