Git, made visual. Stage, diff, commit, and push without memorizing commands. Git shows you exactly what changed and lets you ship it with confidence.

**Highlights**

- See changed files in your working directory
- Stage and unstage with checkboxes; review syntax-highlighted diffs
- Write commit messages and commit only what you select
- Browse commit history, undo the last commit, and push to remote
- Switch between repositories in a click
